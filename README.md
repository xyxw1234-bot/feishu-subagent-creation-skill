# 飞书独立子智能体创建 Skill

这是一个单文件 Hermes Skill，用于统一分发“独立飞书子智能体创建与验收”提示词。

## 使用方式

用户将该仓库中**固定 Release 或固定 commit SHA**对应的 `SKILL.md` 链接发给自己的主智能体，并明确说：

> 请读取并按此 Skill 执行：<固定链接>

不要使用会变化的 `main` 分支链接。

## 用户流程

用户只需要：

1. 扫描飞书官方二维码并命名创建；
2. 在新机器人中发送一次“你是谁？”。

其余步骤由接收 Skill 的主智能体自动执行。

## 说明

本仓库只分发提示词/Skill，不包含插件、服务端、模型凭据、二维码参数或任何用户秘密。
